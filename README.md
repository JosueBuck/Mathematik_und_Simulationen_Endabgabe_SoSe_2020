# Mathematik_und_Simulationen_Endabgabe_SoSe_2020
Dieses Projekt ist als Endabgabe im Rahmen der Vorlesung "Mathematik und Simulationen" entstanden. 
Es handelt sich hierbei um eine Simulation welche die Kollision dreier Massen in einem System darstellt. 
Hierfür wurden physikalisch Realitätsgetreue Gleichungen verwendet, um eine möglichst reale Darstellung zu ermöglichen.
